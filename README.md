## Development

1. Fetch this project
2. Run `npm install`
3. Run `code .` and create `.env` file
4. Write your environment variables based on `.env.example`. Json-server uses port 3000
5. Run `npm start`

## Used technologies

1. React.js
2. Redux
3. Redux-saga
